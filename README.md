
Conway's Game of Life in React
=========

This is a React example for Conway's Game of Life as a code exercise.

* details:
    - TypeScript
    - functional components (not classes)

Rules:
---------

see [here](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life)
